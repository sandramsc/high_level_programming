# High Level Programming

## Introduction
This repository contains programs for the high level programming curriculum at Holberton School.

### Prerequisites

What things you need to install
- **Ubuntu 14.04 LTS** on a Vagrant (Virtualbox) enviroment 
- **shellcheck** to check the style to files bash
- **Git** to clone the repository

## Environment

- Language: Bash script  
- OS: Ubuntu 14.04 LTS
- Style guidelines: Betty to check if the style in files bash is correct


**Step #1** - Clone the project

```bash
$ git clone https://github.com/sandramsc/high_level_programming.git
```
## Usage 

```sh
./[executable file]
```
